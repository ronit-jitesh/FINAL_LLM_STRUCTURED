The Big Change: v5 > v4
Hybrid v5 is now our best system, not v4

We had v4 listed as the best system. That was wrong -- v5's results file had 30 broken rows that were being counted as wrong predictions. Once those were patched, the correct v5 numbers came out significantly better on both matched and mismatched accuracy.

System Matched Acc Mismatched Acc Cost/1k Status
Hybrid v4 90.62% 90.5% $0.007 Best Cost
Hybrid v5 91.0% 92.5% $0.258 Best Overall

Note: v4 is still the right pick if you need the cheapest possible system. v5 is the headline accuracy result.

2
Why v5 Was Underreported
30 GPT-4o rows had parse failures

GPT-4o was producing verbose chain-of-thought outputs in v5, and our label extractor could not parse them. Those rows were being marked as unknown, which counted as wrong. Here is how each broken row was fixed:

19
rows fixed by retrying the GPT-4o API call with a stricter output format
10
rows fixed by majority vote of the 3 DeBERTa predictions (no API cost)
1
row fixed by DeBERTa-base fallback -- all three models disagreed so we used base as tiebreaker

After the fix: 800/800 matched, 400/400 mismatched, 0 unknowns.

Why 51% became 63%

The 51% figure was calculated before the 30 broken rows were patched. Those unknowns were counted as wrong, artificially pulling the number down. After the fix, GPT-4o's true accuracy on the hard 100 is 63%. It is still low (random would be 33%), which still supports the annotation ambiguity story -- just with the correct number.

4
What the v5 Finding Actually Means
87.5%
Unanimous rows
All 3 DeBERTa models agree
95.0% accuracy at $0 API cost
12.5%
Disagreement rows
Models conflict -> GPT-4o
Only 63% accuracy even for GPT-4o

The ensemble gate works differently from confidence gating. Instead of asking "is the encoder confident?", it asks "do all three encoders agree?". The 63% GPT-4o result on the disagreement rows is the key finding -- those cases are not just hard, they appear to be genuinely ambiguous at the annotation level. Even human annotators might disagree on them.

Compare: GPT-4o scores around 85% on the confidence-gated rows in v1-v4. The disagreement rows are a different category entirely.
